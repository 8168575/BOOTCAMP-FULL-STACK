# React Day 2 Assignment (useState Examples + Form Handler)

This Vite + React (JavaScript/JSX) project demonstrates multiple `useState` patterns and includes a complete form handler with validation.

## Features

### ✅ Tabbed interface

- `Counter`
- `Toggle`
- `TextInput`
- `TodoList`
- `ThemeSwitcher`
- `FormHandler`

### ✅ useState examples (React)

- Primitive state (number)
- Boolean toggle
- Controlled text input
- Array state (todo list)
- Object state (theme settings)

### ✅ Form Handler

- Controlled inputs
- Field-level validation
- Error messages
- Submit + Reset

## Project Structure

- `src/app.jsx` - tab UI and example routing
- `src/useStateExamples/`
  - `Counter.jsx`
  - `Toggle.jsx`
  - `TextInput.jsx`
  - `TodoList.jsx`
  - `ThemeSwitcher.jsx`
  - `FormHandler.jsx`
  - `index.js` - exports

## Getting Started

### 1) Install

```bash
npm install
```

### 2) Run dev server

```bash
npm run dev
```

### 3) Build for production

```bash
npm run build
```

### 4) Preview production build

```bash
npm run preview
```

## Notes

- If port `5173` is already in use, Vite will automatically switch to another available port.
